# resume

GitHub Pages で公開している職務経歴書サイト。素の HTML + Tailwind CSS（CDN）で構成。

公開 URL: https://daikimorita.github.io/resume/

## 作業時の注意

- **`index.html` の内容を変更したら、ヘッダーの「最終更新日」を当日の日付に更新するようユーザーに提案すること**（誤字修正など軽微な変更を除く）。ユーザーは更新を忘れがちなので、変更を加えた際は必ずリマインドする。
- Jekyll は使っていない（`.nojekyll` で無効化済み）。`_config.yml` などの Jekyll 設定ファイルを新たに作らないこと。
